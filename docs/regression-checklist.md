# Legacy Todo Regression Checklist

전환 전 Vanilla 앱 기준 동작 체크리스트입니다.

1. 할 일 추가: 공백 입력 시 오류 메시지, 정상 입력 시 목록 최상단에 추가.
2. 할 일 수정: 수정 버튼 진입, Enter 저장, Escape 취소 동작.
3. 할 일 삭제: 삭제 버튼 클릭 시 해당 항목 제거.
4. 완료 토글: 체크박스 클릭 시 완료 상태 반영.
5. 필터: 전체/진행중/완료 필터가 정확히 동작.
6. 통계: 전체/진행중/완료 개수 실시간 반영.
7. 빈 상태: 데이터가 없거나 필터 결과가 없을 때 안내 문구 표시.
8. 길이 제한: 200자 초과 입력 차단 및 에러 표시.
